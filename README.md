# Reto 1: Humano vs Máquina 🐧

Clasificación de especies de pingüinos usando reglas humanas vs Machine Learning.

## Instalación

```bash
pip install -r requirements.txt
```

## Uso

```bash
python main.py
```

## Descripción

El script carga el dataset Palmer Penguins, aplica un clasificador manual basado en reglas y lo compara con un árbol de decisión entrenado automáticamente.
